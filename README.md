# mtproto-up

Automated MTProto proxy setup script 🛠️  
Easily deploy your MTProto proxy with Fake TLS support using Docker.

---

## 🚀 Quick Start

1. Clone the repo 
```bash
git clone https://github.com/rtyfctujucjty/mtproto-up.git
```

2. Run it on your server (with Docker installed):
```bash
bash mtproxy.sh
```
